## Descripción
Este repositorio contiene las soluciones de la práctica de tipología de datos.  
El objetivo es analizar la relación entre la incidencia de Infecciones Respiratorias Agudas (IRAs), la gripe y el polen acumulado, aplicando técnicas de limpieza de datos, exploración, modelado supervisado (Random Forest), clustering (K-Means) y pruebas de contraste de hipótesis.
